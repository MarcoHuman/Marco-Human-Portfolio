# VibeCart — Mobile E-Commerce UI/UX

![VibeCart Preview](../../images/VibeCart.png)

## Description

VibeCart is a modern mobile e-commerce home screen design created in Figma. The interface features a personalized user greeting, a smart search bar, category navigation, daily deals section, and a trending products feed. The design prioritizes a clean, intuitive layout that keeps the user's shopping journey frictionless and engaging.

The project explores **mobile-first UX principles**, balancing visual hierarchy with quick access to key shopping features — a challenge common in real-world e-commerce product design.

## Installation Instructions

This is a Figma design project — no local installation is required.

1. Click the Figma prototype link above.
2. Press the **Play** button in the top-right corner of Figma to launch the interactive prototype.
3. Navigate between screens by clicking/tapping the interactive hotspots.

## Usage

The prototype simulates the user flow of a first-time shopper on the VibeCart app:

- **Home Screen** — Personalized greeting, featured deals, and category filters
- **Search** — Smart search bar with suggested results
- **Product Cards** — Trending items with pricing and quick-add functionality
