# Droewors — Brand Website

![Droewors Preview](../../images/droewors-preview.png)

## Description

A fully designed and developed brand website for a South African droewors (dried sausage) product. The site focuses on high-impact product visuals, brand storytelling, and a responsive mobile-first layout — all built with vanilla HTML, CSS, and JavaScript without any frameworks or external dependencies.

The project required balancing a distinctly South African brand voice with a modern, clean web aesthetic that would appeal to both local and international audiences.

## Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/MarcoHuman/Marco-Human-Portfolio.git
   ```
2. Navigate to the droewors folder:
   ```bash
   cd Marco-Human-Portfolio/BiltongBlvd
   ```
3. Open `Homepage.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

> No build tools, package managers, or servers are required. The site runs entirely from static files.

## Usage

- Navigate through the page sections using the top navigation bar.
- The layout is fully responsive — test it at different screen sizes to see the mobile and desktop breakpoints in action.
- Scroll animations trigger as you move down the page.
