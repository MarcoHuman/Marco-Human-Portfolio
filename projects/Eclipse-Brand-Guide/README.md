# Eclipse Clothing — Brand Identity System

![Eclipse Preview](../../images/Eclipse.png)

## Description

A comprehensive brand identity system for **Eclipse Clothing** — a cosmic-inspired fashion label. The project delivers a full brand guide covering logo design, colour palette, typography hierarchy, brand personality, logo usage rules, digital and print specifications, and apparel mockups.

The brand concept draws on themes of celestial events, darkness, and transformation — positioning Eclipse Clothing as a label for individuals who stand at the edge of light and shadow.
